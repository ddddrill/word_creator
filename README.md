# word_creator
эта программа позволяет составить из букв заданного слова (например ростелеком) новые слова 
реализована она неочень 
прицип основывается на том,что в регулярное выражение забиты буквы слова, которые по которым производить разделение не надо.
если такие буквы имеются, слова делиться на части. Слова, дина которых не превышает 1 будут записываться в список. Далее этот список сверяеться со словарем и записывать новые слова
